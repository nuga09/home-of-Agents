Build a 3D generative AI agent powered by Claude that creates Blender scenes from text prompts or images. The agent should support importing user-provided images and .glb 3D asset files as scene components. It must use an emergent, part-based composition system — meaning complex objects are constructed by generating and assembling atomic parts (e.g. a bicycle emerges from a frame, two tyres, handlebars, a seat, and pedals joined together with spatial constraints). The same emergent logic should apply to organic forms like a female human body, built from parts such as the pelvis, ribcage, abdomen, limbs, skull, and surface skin — assembled using metaballs for seamless organic blending, with correct female proportions (wider hips than shoulders, hourglass waist, fuller thighs). The agent should output a structured scene graph JSON describing each part's shape, dimensions, position, material, and attachment relationships, then compile that into an executable Blender Python (bpy) script run headlessly via CLI. It should support multi-turn refinement so users can iteratively modify the scene through natural language. The full stack should include a Python or Node.js backend, a FastAPI or Express API layer, a React frontend with GLB drag-and-drop and render preview, and Claude's vision API for image understanding.
